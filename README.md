# FlappyAI

A Flappy Bird–style game built in Python using `pygame`, created as a foundation for experimenting with machine learning and reinforcement learning.

## Goals

- Build the game from scratch
- Log gameplay data
- Model player decisions
- Develop an AI agent

## Tech Stack

- Python 3
- pygame
- NumPy
- scikit-learn (later)

## Run

```bash
pip install pygame
python main.py
```

## Vision

Manual play → Data collection → Supervised learning → Reinforcement learning → Autonomous agent
